# NexusCorp IT Asset Management System

**Interactive IT Asset Management System for NEXUS CORP**

This project is developed using **Python Tkinter** and **SQLite**.  
It allows you to add, delete, search, and track IT assets like computers, laptops, printers, routers, and more.

---

## Features
- Front Page with NEXUS CORP branding
- Add / Delete / Search Assets
- Dropdowns for Asset Type and Status
- Dashboard with Progress Bars showing Working, In Repair, and Disposed assets
- View All Assets Table
- Auto-generated Asset IDs
